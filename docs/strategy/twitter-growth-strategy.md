# HUNTER-v2: Twitter Growth Strategy with Agent Teams

## Overview

Claude上で動作するAIエージェントチームを構築し、テック/AIジャンルのTwitterアカウントを0→1→成長させるシステム。

---

## Phase 0: 準備（Week 1）

### Twitter API 取得
- X Developer Portal でアカウント作成
- **Free tier**（月1,500ツイート投稿可能）で開始
- OAuth 2.0 セットアップ
- 必要に応じて Basic ($100/月) にアップグレード

### アカウント設計
- プロフィール設計（AI/テック系で差別化できるペルソナ）
- アイコン・ヘッダー画像準備
- 固定ツイート作成

---

## Phase 1: エージェントチーム設計

### Agent 1: Content Strategist（コンテンツ戦略家）
**役割**: 投稿計画・テーマ選定
- トレンド分析（AI/テック業界の最新ニュース）
- 週間コンテンツカレンダー生成
- コンテンツミックス比率管理
  - 40% 知識共有（TIps, How-to）
  - 25% ニュース・トレンド解説
  - 20% 意見・考察
  - 15% エンゲージメント促進（質問、投票）

### Agent 2: Content Creator（コンテンツ制作者）
**役割**: ツイート文面の生成
- 戦略家の計画に基づいてツイート原稿作成
- スレッド形式の長文コンテンツ
- フック（最初の一文）の最適化
- ハッシュタグ選定

### Agent 3: Engagement Analyst（エンゲージメント分析者）
**役割**: パフォーマンス分析・改善提案
- 投稿パフォーマンス追跡（いいね、RT、リプライ数）
- 最適投稿時間帯の分析
- A/Bテスト結果の分析
- フォロワー増加トレンド分析

### Agent 4: Community Manager（コミュニティ管理者）
**役割**: エンゲージメント・コミュニティ構築
- リプライ候補の生成
- フォローすべきアカウントの提案
- コラボ/引用RT機会の発見
- メンション・DM対応案の作成

### Agent 5: Orchestrator（オーケストレーター）
**役割**: エージェント間の調整・ワークフロー管理
- 各エージェントのタスク割り当て
- スケジュール管理
- 品質チェック（ブランドボイス一貫性）
- 人間への承認リクエスト管理

---

## Phase 2: 成長戦略（0→1,000 フォロワー）

### Week 1-2: 基盤構築
- プロフィール最適化
- 初期コンテンツ投稿（1日3-5ツイート）
- 業界インフルエンサー50人フォロー&エンゲージメント
- 自己紹介スレッド作成

### Week 3-4: エンゲージメント強化
- 人気アカウントへの価値ある引用RT
- コミュニティへの積極参加
- ハッシュタグ戦略の実行（#AI, #LLM, #ChatGPT等）
- 初スレッド投稿（バズ狙い）

### Month 2-3: 加速
- バイラルコンテンツフォーマットの確立
- 他アカウントとのコラボ
- 定期的なまとめスレッド
- データドリブンな投稿時間最適化

---

## Phase 3: 成長戦略（1,000→10,000 フォロワー）

- ニッチ内での権威確立
- オリジナルリサーチ/分析コンテンツ
- スペース（音声機能）活用
- メールリスト連携
- クロスプラットフォーム展開

## 技術スタック

| コンポーネント | 技術 |
|---|---|
| エージェント基盤 | Claude API (claude-sonnet-4-6) |
| オーケストレーション | Python + Claude Agent SDK |
| Twitter連携 | tweepy (X API v2) |
| データストア | SQLite |
| スケジューラ | APScheduler / cron |
| ダッシュボード | Streamlit (optional) |
| 設定管理 | YAML |

---

## コンテンツカテゴリ例（テック/AI）

1. **AI最新ニュース速報** - 新モデルリリース、論文紹介
2. **実践Tips** - プロンプトエンジニアリング、API活用法
3. **比較・分析** - モデル比較、ツール比較
4. **チュートリアル** - ステップバイステップガイド（スレッド）
5. **業界考察** - AIの未来、倫理、ビジネスインパクト
6. **ツール紹介** - 便利なAIツール、ライブラリ紹介
7. **コード共有** - コードスニペット、使い方
